## Lightcurve Fourier Decomposition
Direct Fourier decomposition for multi-band Cepheid variable light curves
- Find initial period by Lomb-Scargle method (implemented by '[gatspy]'(https://www.astroml.org/gatspy/)(VanderPlas et al. 2018))
- Iterative penalized least-square fitting
- Multi-band compilation of Fourier parameters are also available 

This code is adopted to analyze the [OGLE IV](<href="https://ogledb.astrouw.edu.pl/~ogle/OCVS/">) Cepheid/RR Lyrae variables.
